# Netflix Clone Project 
This project is a clone of Netflix's web application and is developed using React. The project has separate categories in which movies coming from the API are listed. Each category contains a field containing information about movies of that genre. Additionally, users can switch between movies with a slider.

# Features 
- Listing movies from API.
- Filter by categories.
- Ability to switch between movies with slider.
- Opening the detail page when clicking on the movies.
- The detail page includes information such as the film's actors, income, country and language.
- Opportunity to watch trailer on detail page.

# Libraries Used 

- Axios: Used for API requests.
- React Router DOM: Used for page redirects and deep links.
- React Player: Used to play trailers.
- Redux: Used for state management.
- Redux Thunk: Used for Redux actions.
- Bootstrap: Used for basic styling and editing.

# How to Operate 

1. Clone the repo to your local machine.
2. Run npm install to install the required dependencies.
3. Run the npm start command to start the development server.
4. Go to localhost:3000 in your browser.
